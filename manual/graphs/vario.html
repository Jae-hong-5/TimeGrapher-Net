5. Vario

레이트와 진폭의 안정도를 게이지와 통계로 보여 주고, "기록해도 될 만큼 안정적인지"를 색으로 판정합니다.

Shows rate and amplitude stability as gauges + stats, and gives a colour verdict on whether the watch is "stable enough to record".

Mostra a estabilidade da taxa e da amplitude como medidores + estatísticas e dá um veredicto por cor sobre se o relógio está "estável o suficiente para registrar".

용도 · Purpose · Finalidade

측정값이 얼마나 퍼져 있는지(표준편차)를 보고, 지금이 신뢰할 만한 측정 시점인지 즉시 판단합니다.

Reads how spread out the readings are (standard deviation) so you can tell at a glance whether the watch is settled enough to trust.

Lê o quanto as leituras estão dispersas (desvio padrão) para que você possa identificar rapidamente se o relógio está estabilizado o suficiente para ser confiável.

무엇이 보이나 · What's shown · O que é exibido

  • Error Rate 게이지 — 가로축 Error Rate(s/d) −10~+10. 음영 띠 = 허용 범위, 파란 선 = 최소·최대, 빨간 선 = 평균, 점선 = 현재값. 곁에 min / max / avg / current 라벨.Error Rate gauge — axis = Error Rate (s/d) −10…+10. Shaded band = acceptable range; blue lines = min/max; red line = mean; dashed = current. Labels min / max / avg / current.Medidor de Error Rate — eixo = Error Rate (s/d) −10…+10. Faixa sombreada = intervalo aceitável; linhas azuis = min/max; linha vermelha = média; tracejado = atual. Rótulos min / max / avg / current.
  • 진폭 게이지 — 같은 구성, 건강범위 270–300°.Amplitude gauge — same layout, healthy band 270–300°.Medidor de Amplitude — mesmo layout, faixa saudável de 270–300°.
  • 상단 요약 — 상태 칩(녹/황/적) + 종합 판정·조치 텍스트(예: "Overall: OK · No service indicated", "Overall: ALERT · Service diagnosis indicated").Top summary — status chips (green/yellow/red) + overall verdict with action (e.g. "Overall: OK · No service indicated", "Overall: ALERT · Service diagnosis indicated").Resumo superior — chips de status (verde/amarelo/vermelho) + veredicto geral com ação (ex.: "Overall: OK · No service indicated", "Overall: ALERT · Service diagnosis indicated").
  • 통계표 — 각 게이지 아래 Min · Max · Spread · Average · Sigma · Current.Stats grid — under each gauge: Min · Max · Spread · Average · Sigma · Current.Grade de estatísticas — abaixo de cada medidor: Min · Max · Spread · Average · Sigma · Current.

Spread(최대−최소)와 Sigma(표준편차)가 작을수록 좋습니다. 장기 기록을 스크럽하려면 Long-Term 탭의 리뷰 바를 사용하세요.

Smaller Spread (max−min) and Sigma (std dev) are better. To scrub long-run history, use the review bar in the Long-Term tab.

Quanto menores o Spread (max−min) e o Sigma (desvio padrão), melhor. Para percorrer o histórico de longo prazo, use a barra de revisão na aba Long-Term.
