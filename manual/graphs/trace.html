3. Trace

측정이 진행되는 동안 레이트와 진폭이 흔들리지 않고 유지되는지 시간축으로 추적합니다.

Tracks whether rate and amplitude hold steady over the elapsed run — the "is it keeping its settings?" view.

용도 · Purpose

한 자세에서 시계를 일정 시간 두고, 시간이 가도 정확도와 진폭이 일정한지(또는 점점 처지는지) 봅니다.

Leave the watch in one position for a while and see if accuracy and amplitude stay flat (or sag over time).

진폭 (아래)Elapsed (s)Amplitude (°)선 + 270–300° 건강범위 음영 띠.Line + shaded 270–300° healthy band.

경고 배너(위) — 레이트가 −1 s/d보다 느리거나 진폭이 270–300°를 벗어나면 ⚠와 함께 "Watch is running late (X s/d)" 또는 "Amplitude X° outside normal range 270–300°"가 표시됩니다.

Alert banner (top) — if rate runs slower than −1 s/d, or amplitude leaves 270–300°, a ⚠ shows "Watch is running late (X s/d)" or "Amplitude X° outside normal range 270–300°".

요약 푸터 — "RATE avg X s/d / last 60.0s Y s/d | AMP avg Z° / last 60.0s W°"(전체 평균과 최근 60초 평균).

Summary footer — "RATE avg / last 60.0s … | AMP avg / last 60.0s …" (since-start vs. last-60s average).

읽는 법 · How to read

  • 좋음: 두 선이 평평하게 유지 — 레이트는 0 근처, 진폭은 음영 띠 안.Good: both lines stay flat — rate near 0, amplitude inside the shaded band.
  • 주의: 진폭이 시간이 갈수록 처지면 윤활/파워리저브 문제 신호일 수 있습니다.Watch out: amplitude drifting down over time can signal lubrication / power-reserve issues.
  • ⚠ 배너가 뜨면 사양을 벗어난 것 — 일시정지 후 원인을 살피세요.A ⚠ banner means out of spec — pause and investigate.

짧은 시간의 안정도 판정은 Vario, 몇 시간 단위의 장기 추세는 Long-Term이 더 적합합니다.

For a short-window stability verdict use Vario; for multi-hour trends use Long-Term.
